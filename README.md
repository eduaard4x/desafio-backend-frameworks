# desafio-backend-frameworks
Objetivo: Evidenciar o conhecimento prático sobre a Arquitetura em Camadas e a Classificação por Linguagem através do "Bootstrap" (inicialização) de dois projetos back-end diferentes.

#Comparação
Express: simples e rápido
Spring Boot: mais completo e organizado

#Arquitetura
Model: dados
Controller: requisições
Routes/Services: ligação entre camadas

Os dois frameworks seguem a mesma lógica de organização, mudando apenas a linguagem e complexidade.
